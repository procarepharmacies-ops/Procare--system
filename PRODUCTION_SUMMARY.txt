🚀 QUICK DEPLOYMENT GUIDE

1. CREATE SLACK BOT (5 min)
   → https://api.slack.com/apps
   → "Create New App" → "From scratch"
   → Name: ProCare Pharmacy

2. CONFIGURE OAUTH SCOPES (2 min)
   → OAuth & Permissions
   → Add: chat:write, chat:write.public
   → Click "Install to Workspace"

3. COPY BOT TOKEN (1 min)
   → Copy "Bot User OAuth Token" (xoxb-...)
   → Edit .env: SLACK_BOT_TOKEN=xoxb-...

4. INSTALL DEPENDENCIES (2 min)
   pip install -r requirements.txt

5. TEST INTEGRATION (2 min)
   curl http://localhost:5000/api/slack/test

6. SCHEDULE AUTOMATION (5 min)
   Windows Task Scheduler: Daily at 7:00 AM
   Command: python tools/hermes_slack_sync.py
